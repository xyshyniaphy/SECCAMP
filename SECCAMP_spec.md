# SECCAMP（プライベートキャンプ場探索AIエージェント）仕様書

**版:** 1.0  
**作成日:** 2025年12月17日  
**目的:** キャンピングカー・プライベートキャンプ場に最適な土地を日本全国から自動検索・分析するAIエージェント

---

## 1. プロジェクト概要

### 1.1 システム名称
**SECCAMP** (Secure campsite search and Community-aware camping land Matcher and Analyzer Platform)

### 1.2 目的・背景
- 日本におけるプライベートキャンプ場（野営地）需要の拡大
- キャンピングカーの所有者増加に伴う適切な土地探しの困難性
- 既存不動産サイトでは「キャンプ場適地」という条件検索が限定的である
- 複数の不動産情報源の統一的な検索・分析の必要性

### 1.3 主な特徴
- **自動スクレイピング機能:** 複数の日本主要不動産サイトから一括検索
- **インテリジェントフィルタリング:** キャンプ適地の複合的な条件判定
- **リアルタイムマッピング:** 候補地の可視化と周辺情報の表示
- **AI分析エンジン:** 写真・敷地図からキャンプ適性を判定

## 2. スコープ・対象

### 2.1 対象ユーザー
1. **プライベートキャンプ場の購入・賃借検討者**
2. **キャンピングカーユーザー（定住地探し）**
3. **グラマーピング・ゲストハウス開業者**
4. **自給自足/オフグリッド生活検討者**
5. **小規模キャンプ場・グランピング施設経営者**

### 2.2 対象地域
- **第1優先:** 関東地方（東京・神奈川・埼玉・千葉・栃木・茨城・群馬）
- **第2優先:** 中部地方（山梨・長野・岐阜・愛知・静岡）
- **第3優先:** その他地域（拡張時対応）

### 2.3 対象物件タイプ
- 山間部の土地（純山林→NG。人が活動できる空地）
- 更地（現状が空地の状態）
- 古民家付き土地
- 廃屋付き土地
- その他（納屋・倉庫付きなど）

```

- 最寄り駅名・距離（km）

【立地条件】
- 標高（m）
- 傾斜度（%）
- 周辺環境（山林/田畑/住宅地/混在等）
- 周辺人口密度（人/km²）
- 隣接住宅までの距離（最短、平均）

【ユーティリティ情報】
- 水道接続可否（上水/簡易水道）
- 電気接続可否（60A/100A/その他）
- ガス（都市ガス/LPガス）
- 下水（公共下水/浄化槽/汚水処理）
- 通信（光回線/5G/4G電波強度）

【利用規制】
- 農地区分（農地該当/雑種地/宅地等）
- 建築可否（建築確認必要/規制あり等）
- 開発許可区域（市街地調整区域/第1種低層等）
- 景観計画地域（該当/非該当）
- 防火地域指定（有/無）
- 保安林指定（有/無）

【メディア】
- 写真URL（複数枚、遠景/近景/周辺状況）
- 敷地図/位置図URL
- 動画URL（ある場合）

【メタデータ】
- 掲載元サイト名
- 掲載日時
- 更新日時
- 問い合わせ先（電話/メール）
```

#### 3.2.2 高度なフィルター（システム自動判定）

**キャンプ適性スコア計算**

```
キャンプ適性スコア = Σ（各条件のスコア × 重み係数）

・広さ適性: (面積 / 理想面積) × 25点 → 最大25点
・隣近所配慮: 1 - (民家数 / 許容数) × 20点 → 最大20点
・道路適性: (道路幅員 / 最小幅員) × 20点 → 最大20点（上限20）
・生活利便性: (都市機能数 / 最大値) × 15点 → 最大15点
・景観点: (山林率) × 10点 → 最大10点
・アクセス: 1 - (移動時間 / 最大許容時間) × 10点 → 最大10点

合計: 100点満点

判定基準:
  80点以上: ★★★★★ 極めて適正
  60～79点: ★★★★ 高度に適正
  40～59点: ★★★ 中程度に適正
  20～39点: ★★ 要検討
  <20点: ★ 不向き
```

#### 3.2.3 除外ルール（AND条件）

以下に**すべて**該当する場合は除外：
- 道路幅員 < 3.5m
- 隣接住宅までの距離 < 200m かつ 周辺人口密度 > 100人/km²
- 農地法規制 かつ 転用許可不可の土地
- 市街化調整区域 かつ キャンプ場営業許可基準不適合
- 土地面積 < 500㎡

### 3.4 AI分析エンジン

#### 3.4.1 画像認識・分析

**入力:** 敷地写真・位置図

**分析項目:**

| 項目 | 手法 | 出力 |
|------|------|------|
| 地表面タイプ判定 | CNN（Convolutional Neural Network） | 更地/草地/山林/水面 等 |
| 傾斜度推定 | 画像からの勾配分析 | %単位での推定値 |
| 方角判定 | メタデータ・画像コンテキスト | 南向き/北向き 等 |
| 樹木密度推定 | セマンティックセグメンテーション | 密度0～100% |
| 周辺建物検出 | Object Detection | 民家数、距離推定 |
| 道路幅員推定 | 画像からの距離推定 | m単位 |
| 隣接施設判定 | 写真コンテキスト分析 | 発電所/工場/墓地 等 |

#### 3.4.2 テキスト解析

**入力:** 物件情報テキスト、説明欄、口コミ

**分析項目:**

| 項目 | 手法 | 出力 |
|------|------|------|
| ネガティブ要因抽出 | NLP（自然言語処理） | 騒音/臭気/浸水リスク 等 |
| ポジティブ要因抽出 | テキスト感情分析 | 静寂/景観/アクセス 等 |
| 法律用語認識 | 固有表現認識（NER） | 農地法/建築基準法等の適用状況 |
| 交渉余地判定 | テキスト内容分析 | 交渉可能性スコア |

#### 3.4.3 時系列分析

- **価格トレンド:** 同一物件の価格推移
- **掲載期間分析:** 長期掲載物件は要注意フラグ
- **成約速度:** 同地域の平均掲載期間との比較

```sql
  listing_date DATETIME,             -- 掲載日

  campsite_score DECIMAL(5, 2),      -- キャンプ適性スコア
  created_at DATETIME,
  updated_at DATETIME,
  INDEX idx_location (location_pref, location_city),
```

```python
# 疑似コード
def scrape_properties():
    sources = [
        ('athome', 'https://www.athome.co.jp/tochi/'),
        ('suumo', 'https://suumo.jp/tochi/'),
        ('ieichiba', 'https://www.ieichiba.com'),
        # ...
    ]
    
    for source_name, base_url in sources:
        try:
            driver = webdriver.Chrome()  # Seleniumで動的ページ対応
            properties = extract_properties(driver, base_url)
            
            for prop_data in properties:
                normalized_prop = normalize_data(prop_data)
                detect_duplicates(normalized_prop)  # 重複排除
                save_to_db(normalized_prop)
                
        except Exception as e:
            log_error(f"Error scraping {source_name}: {e}")
        finally:
            driver.quit()
    
    # キャッシュ更新・スコア再計算
    recalculate_all_scores()
```

#### 4.4.3 重複検出ロジック

```
1. 住所 + 面積 + 価格で候補判定（Fuzzy Match 90%以上）
2. 複数の掲載元で同一物件の場合は統合
3. 更新日時が新しいデータで上書き
```

### 4.5 AI/ML実装

#### 4.5.1 スコアリングモデル

```
入力層:
  - 広さスコア (0-25)
  - 隣近所配慮スコア (0-20)
  - 道路適性スコア (0-20)
  - 生活利便性スコア (0-15)
  - 景観スコア (0-10)
  - アクセススコア (0-10)

重み調整:
  - ユーザー入力「隔離度重視」→ 隣近所配慮を1.5倍
  - 「アクセス重視」→ アクセススコアを1.3倍
  
出力層:
  - 最終スコア (0-100)
  - 信頼度スコア (0-100)
```

#### 4.5.2 画像認識モデル

- **プリトレーニング:** ResNet50 / EfficientNet（ImageNet）
- **ファインチューニング:** キャンプ場適性判定用カスタムデータセット
- **出力:** 各要素（更地度、樹木密度、傾斜度、民家有無 等）のスコア
